# Statistics Practice

A Python data analysis and visualization practice project.

This repository demonstrates a typical real-world data analysis workflow using pandas and matplotlib.

## Project Structure

```text
.
├── data/
│   └── sales.csv
├── outputs/
│   └── .gitkeep
├── src/
│   └── analysis.py
├── requirements.txt
├── .gitignore
└── README.md
```

## Features

- CSV data loading
- Data cleaning
- Descriptive statistics
- Revenue analysis
- Grouped aggregation
- Correlation analysis
- Data visualization
- Export cleaned data

## Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run analysis:

```bash
python src/analysis.py
```

Generated charts and cleaned data will be stored in the `outputs/` folder.

## Tech Stack

- Python
- pandas
- matplotlib
- seaborn
